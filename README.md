# Higher Education ERM Demo (COSO + RCSA)

This project demonstrates a COSO-aligned Enterprise Risk Management (ERM) framework using RCSA methodology with synthetic data.

## Features
- Risk Register (RCSA-based)
- Inherent and Residual Risk Scoring
- Control Effectiveness Modeling
- Higher Education Finance Risk Scenarios

## Disclaimer
All data in this project is synthetic and for demonstration purposes only.
